# 无人超市管理系统 python890

本项目是基于 Python3.7、Flask 框架和 MySQL5.7 的无人超市管理系统，前端采用 Vue 技术栈。

## 项目概述

无人超市管理系统旨在通过信息化手段，高效管理超市的日常运营工作，包括用户管理、商品管理、进货管理以及订单处理等。

## 技术栈

- 后端：Python3.7、Flask
- 数据库：MySQL5.7
- 前端：Vue

## 功能模块

### 系统角色

- 管理员
- 员工
- 用户

#### 管理员功能

- 用户管理
- 员工管理
- 商品分类管理
- 超市商品管理
- 商品进货管理
- 供应商管理
- 系统管理
- 订单管理
- 用户资料管理

#### 员工功能

- 超市商品操作
- 商品进货操作
- 订单管理
- 用户资料操作

#### 用户功能

- 商品浏览
- 购物车管理
- 我的订单
- 个人中心

## 运行环境

- 后端：Python3.7 环境搭建，依赖 Flask 框架和 MySQL5.7 数据库
- 前端：基于 Vue 的浏览器运行环境

## 部署步骤

1. 搭建 Python3.7 开发环境
2. 安装 Flask 框架和 MySQL5.7 数据库
3. 配置数据库连接
4. 部署前端 Vue 代码到 Web 服务器
5. 启动后端 Flask 应用
